# Order service

## Description

A service that exposes a RESTFul API for managing orders

## Project setup

```bash
# install all dependencies
$ pnpm install
```

## Compile and run the project

```bash
# development watch mode
$ pnpm run dev

# debug mode
$ pnpm run start:debug

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
